SNS 이용자 분류 제출 완료

 문제의 입력값은 각 언어의 표준입력(stdin) 함수를, 출력값은 표준출력(stdout) 함수를 사용해주세요.

(작성한 코드의 시간복잡도(Time Complexity)와 공간복잡도(Space Complexity)를 Big-O 표기법을 이용해서, 코드 상에 주석으로 기입해주세요. 채점시 참고합니다. 단, 코드를 작성한 경우만 기입하면 됩니다. 주석만 있고 구현된 코드가 없는 답안은 채점하지 않습니다.)

서로 친구 관계인 사용자들을 그룹화 하려 한다. 서로 친구 관계에 있는 사람들을 모두 연결하여 하나의 그룹으로 만들었을 때 (직접적인 친구가 아니더라도 친구를 통하여 연결이 되어있다면 하나의 그룹으로 본다) 주어진 입력을 통하여 총 몇개의 그룹으로 나누어 지는지 출력하는 프로그램을 작성하시오.

입력형식

  • 첫째 줄엔 친구 관계 묶음의 갯수 N이 주어지며 (2 <= N <= 100,000)
  • 둘째 줄 부터 N개의 줄에 걸쳐 서로 친구 관계에 있는 사람의 번호(x y) 묶음이 주어진다 (1 <= x, y <= 100,000)

출력형식

  • 그룹의 수

구름두이노 크롬 앱

아두이노를 연결하여 실행하기 위해서는 "구름두이노"의 설치가 필요합니다.

오른쪽 아래에 있는 "설치하기" 버튼을 통해 설치해주시기 바랍니다.

(현재는 크롬 브라우져만 지원하고 있으나, 추후 파이어폭스(Firefox)나 엣지(Edge) 등까지 확장할 예정입니다.)

해당 브라우져에서는 이미 설치가 되어있습니다.

"구름두이노"을 설치하셨다면, 앱과 연결을 해야합니다.

오른쪽 아래에 있는 "연결하기" 버튼을 통해 앱과 연결해주시기 바랍니다.

연결이 되셨다면, 아두이노 보드를 PC와 연결하신 후, "보드 불러오기" 오른쪽에 있는 을 클릭하여 아두이노 보드 리스트를 불러옵니다.

USB로 연결하신 경우, 리스트에서 "tty.usb"로 시작되는 보드를 확인하실 수 있으니 해당 보드를 선택해주시길 바랍니다.
